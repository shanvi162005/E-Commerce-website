# 🛒 My E-Commerce Website

Build a full-stack e-commerce platform that allows users to browse products, add them to a shopping cart, and complete a checkout process. Users will be able to create accounts, log in, and manage their cart. Admins can manage products.

## ✨ Key Features

* **Secure Auth**: Implemented JWT-based authentication to protect user data and maintain sessions.
* **Product Catalog**: Responsive grid interface that fetches product data directly from the MongoDB database.
* **Real-time Cart**: Users can manage quantities and remove items, with total prices calculated dynamically.
* **Responsive UI**: Modern dark-themed design optimized for desktop and mobile.

## 🛠 Tech Stack

* **Frontend**: React, Axios, HTML, CSS (Flexbox/Grid)
* **Backend**: Node.js, Express.js
* **Database**: MongoDB & Mongoose
* **Authentication**: JWT (JSON Web Tokens), bcryptjs
